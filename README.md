# PORTFOLIO BASIC - ADMIN PANEL WITH LARAVEL 8

How to create Portfolio - admin panel with Laravel 8.

# Stack

* Laravel 8
* MySql
* Artisan
* Composer
* Bootstrap
* Paquete Laravel UI
* Generador Automático de CRUD
* Template Freelancer Bootstrap https://startbootstrap.com/theme/freelancer

## The Commands 
* to create project -->  composer create-project laravel/laravel:^8.5.16 portafolio-admin-laravel8 --ignore-platform-reqs 
* php artisan make:migration create_proyectos_table (para crear la migración Proyectos)
* php artisan migrate (para pasar las tablas a la base de datos)
* composer require laravel/ui (creamos la autenticación)
* php artisan ui bootstrap --auth (instalamos bootstrap)
* npm install (para compilar)
* npm run dev (para compilar también)
* composer require ibex/crud-generator --dev (instalara generador automático de crud)
* php artisan vendor:publish --tag=crud (para acceder a la configuración de la librería)
* php artisan make:crud proyectos (para crear el crud referencia a la tabla proyectos)
* php artisan make:controller PortafolioController (creamos el controlador Portafolio)

## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

- [Simple, fast routing engine](https://laravel.com/docs/routing).
- [Powerful dependency injection container](https://laravel.com/docs/container).
- Multiple back-ends for [session](https://laravel.com/docs/session) and [cache](https://laravel.com/docs/cache) storage.
- Expressive, intuitive [database ORM](https://laravel.com/docs/eloquent).
- Database agnostic [schema migrations](https://laravel.com/docs/migrations).
- [Robust background job processing](https://laravel.com/docs/queues).
- [Real-time event broadcasting](https://laravel.com/docs/broadcasting).

Laravel is accessible, powerful, and provides tools required for large, robust applications.
